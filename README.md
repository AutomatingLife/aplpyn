# aplpyn
Welcome to aplpyn (pronounced "apple pen"), a Python CLI for APL.

## Features

* Full APL support
* APL ASCII completion
* APL autocomplete
* Making APL functions python objects
* File I/O support
* Stateful and Stateless APL sessions
* No APL interpreter required
